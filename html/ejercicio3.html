Definicion de 3 palabras relacionadas con sistemas.

Algoritmo: Conjunto de instrucciones o pasos secuenciales y finitos que se siguen para resolver un problema o realizar una tarea. Es la base de la programación y de los procesos en los sistemas informáticos.

Red: Conjunto de dispositivos interconectados que permiten la comunicación entre ellos, como computadoras, servidores, y otros dispositivos. Las redes pueden ser locales (LAN) o extenderse globalmente (como Internet).

Base de datos: Sistema organizado para recopilar, almacenar y gestionar datos de manera eficiente. Permite la manipulación y consulta de grandes volúmenes de información de forma rápida y segura.
